### 1.create .env file 
Add following environment variables

```
OPENAI_API_KEY= 
LANGCHAIN_API_KEY= 
LANGCHAIN_TRACING_V2=true 
LANGCHAIN_PROJECT=ReAct agent executor 
TAVILY_API_KEY=
```

### 2. Install packages

run `poetry install` 

### 3. Open project in your fav editor

Debug main.py in your favourite project editor
